/* ЗАДАНИЕ 1
 * В базе данных Redis подберите коллекцию для подсчета посещений с определенных IP-адресов.
 *
 * Решение: 
 * Используем сортированное (ранжированное) множество. Элементом множества будет
 * выступать строка с ip-адресом, а ранг элемента будет показывать количество посещений.
 * Команда ZINCRBY добавляет к рангу элемента указанное число, если элемента или множества
 * к моменту выполнения команды не существовало, они будут созданы. Получить количество 
 * посещений можно комнадами ZRANGE (для нескольких элементов) или ZRANK (для указанного)
 *
 * Ниже приведены команды redis-cli и выводы консоли по результатам выполнения этих команд.
 */

127.0.0.1:6379> keys *
(empty list or set)
127.0.0.1:6379> ZINCRBY ip 1 '10.15.28.1'
"1"
127.0.0.1:6379> ZINCRBY ip 1 '10.15.28.2'
"1"
127.0.0.1:6379> ZINCRBY ip 1 '10.15.28.1'
"2"
127.0.0.1:6379> ZINCRBY ip 1 '10.15.28.1'
"3"
127.0.0.1:6379> ZINCRBY ip 1 '10.15.28.3'
"1"
127.0.0.1:6379> ZRANGE ip 0 -1 WITHSCORES
1) "10.15.28.2"
2) "1"
3) "10.15.28.3"
4) "1"
5) "10.15.28.1"
6) "3"
